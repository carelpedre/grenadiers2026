# Photo drop folder

Drop photos here using the exact filenames listed below. The site picks them up automatically — no code changes needed. Refresh the browser to see them.

## Filename rules
- **All lowercase**
- **No accents** (Sébastien → sebastien, Sainté → sainte, Adé → ade, Casimir's é → casimir)
- **Hyphens between words** (no spaces, no underscores)
- **`.jpg` extension** (`.webp` and `.png` also work)
- Aim for **at least 1200px on the long side** for hero images, 600px for player cards

## Tips for sourcing photos

- **Action shots** beat posed portraits — players in their club kit or Haiti kit, mid-action
- **Photos of just the player** (cropped, head + shoulders or thigh-up)
- **High resolution** — the site downscales automatically
- For stadium and historical photos, **landscape orientation** works best (16:9 or 21:9 aspect)
- For player cards, **portrait or square** crops work well — the system handles both

## When a photo doesn't show up

1. Double-check the filename — must match **exactly** (case, hyphens, no spaces)
2. Confirm it's saved in `public/images/photos/` (not just `public/images/`)
3. Hard refresh the browser: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
4. If it still doesn't show, the Haiti red/blue placeholder will display instead — site stays functional
